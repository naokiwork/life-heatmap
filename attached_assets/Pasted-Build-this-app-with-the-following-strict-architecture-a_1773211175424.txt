GOAL

CLOUDFLARE COMPATIBILITY RULES

That means:
- Do not build around Replit deployment assumptions

- Prepare deployment configuration files needed for Cloudflare
- If framework choice matters, choose one with smooth Cloudflare compatibility
- Avoid unnecessary server dependencies that make Cloudflare deployment difficult

SECRETS AND SECURITY RULES

- Use environment variables everywhere

- Add rate limiting or at least leave a clean extension point for rate limiting
- Prevent duplicate webhook processing
- Use least-privilege architecture where possible
- Never expose service role keys to the client
- Clearly separate public env vars and server-only secrets
